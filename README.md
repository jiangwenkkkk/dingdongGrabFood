# dingdongGrabFood
叮咚抢菜
##
1. 利用Anaconda程序创建一个python3.7的环境
3. 执行python setup.py install
4. 在windows上打开微信客户端，打开微信小程序，进入到叮咚买菜
5. 将小程序的客户端放到屏幕左上角
6. 执行python grabfood.py进入抢菜
7. 按q退出
